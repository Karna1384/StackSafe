# StackSafe

A small team project for learning Git and GitHub, step by step.

## About this project

StackSafe is a tiny web page built by three teammates: **Alex**, **Priya**, and **Sam**.
It doesn't do much yet, just a simple welcome message, but that's the point.
This repo exists so you can learn Git the same way a real team would use it.

## For students

Do not edit this file directly. Just follow the tutorial your instructor gave you.

When you're ready, click the green **Code** button above, then **Codespaces**, then **Create codespace on main**.
